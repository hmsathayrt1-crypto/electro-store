# ⚡ متجر الأدوات الكهربائية والإلكترونية

متجر إلكتروني عربي متكامل لبيع الأدوات الكهربائية والإلكترونية.

## التقنيات
- PHP 8.x + MySQL 8
- HTML5/CSS3/JavaScript + AJAX
- بوت ذكي مدمج

## المتطلبات
- PHP 8.0+
- MySQL 8.0+
- Apache/Nginx

## التثبيت
1. استيراد `sql/schema.sql` ثم `sql/seed.sql`
2. نسخ `.env.example` إلى `.env` وتعديل الإعدادات
3. تعديل `config/database.php` ببيانات الاتصال

## بيانات الدخول
- **مدير:** admin@store.com / admin123
- **عميل:** سجل حساب جديد

## المميزات
- تصفح وبحث وفلترة منتجات
- سلة مشتريات ذكية (AJAX)
- 3 طرق دفع (نقداً، محفظة، بوابة)
- محفظة إلكترونية
- بوت ذكي للمحادثة
- لوحة تحكم كاملة للمدير
- تقارير مبيعات
